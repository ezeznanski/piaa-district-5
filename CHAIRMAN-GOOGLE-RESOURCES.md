# Chairman-managed Google resources

The website is prepared to use Google Docs and Sheets that update without replacing files on GitHub.

## Football pilot folder

Create this structure in a District V Google Shared Drive:

- PIAA District V Sports
  - Football
    - 2026-2027 Football Handbook (Google Doc)
    - Class A Weekly Rankings (Google Sheet)
    - Class AA Weekly Rankings (Google Sheet)
    - Class A Playoff Bracket (Google Sheet)
    - Class AA Playoff Bracket (Google Sheet)
    - Football Scores and Schedule (Google Sheet)

## Permissions

1. Give the Football chairman **Editor** access to the Football folder only.
2. Keep the District V account as the owner/manager.
3. Do not grant the chairman edit access to the website repository.
4. Publish each public file using **File → Share → Publish to web**.
5. Use a published link, not an editor link, on the public website.
6. Confirm the published link opens in a private/incognito browser without signing in.

## Connect the files

The Football links are stored in `resource-links.js`. Replace each legacy District V URL with its corresponding published Google URL. This connection is made only once. Afterward, chairman edits to the source Google file flow through to the published resource automatically.

The original links remain as safe fallbacks until the Google files are created and reviewed.
